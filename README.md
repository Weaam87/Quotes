# In this app , I tried to show my skills in the following subjects :

1. Get data from the internet.
2. Push Notifications.
3. Create custom JSON file and uploaded it to the server.
4. Adaptive layouts , use SlidingPaneLayout to adaptable to different screen sizes.
5. Establish web service request Using Retrofit,Moshi,and Coil libraries.
6. Add custom back navigation.
7. Using BindingAdapter to create custom setter for properties.
8. Add an enum class to represent all the available connection statuses in the UI.
9. Shared ViewModel.
10. Using coroutine to launch functions in the background.
11. Jetpack Navigation component.
12. Material design.
13. Using one activity with multiple fragments.
14. LiveData.
15. Data binding.
