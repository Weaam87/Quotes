# In this app , I tried to show my skills in the following subjects :

1. Get data from the internet.
2. Create custom JSON file and uploaded it to the server.
3. Establish web service request Using Retrofit,Moshi,and Coil libraries.
4. Using BindingAdapter to create custom setter for properties.
5. Add an enum class to represent all the available connection statuses in the UI.
6. Shared ViewModel.
7. Using coroutine to launch functions in the background.
8. Jetpack Navigation component.
9. Material design.
10. Using one activity with multiple fragments.
11. LiveData.
12. Data binding.
